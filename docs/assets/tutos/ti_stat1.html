Données statistiques - calculatrice TI

Saisie des données

  • Entrer dans le menu STAT.
  • Appuyer sur F6 pour faire apparaître les onglets GRPH et CALC.
  • Entrer la série de données (xi) dans la première colonne.
  • Entrer les effectifs (s’il y en a) dans la deuxième colonne.

Affichage des indicateurs statistiques

  • Sur l’écran STAT, appuyer sur F2 puis sur F6.
  • Séries sans effectifs : aller sur 1Var Freq, puis appuyer sur F1.
    Séries avec effectifs : aller sur 1Var Freq, puis appuyer sur F2, 2 puis EXE.
  • Appuyer sur EXIT, puis F1.
  • Avec les flèches haut et bas, faire défiler la liste d’indicateurs.

Afficher un diagramme en boîte

  • Sur l’écran STAT, appuyer sur F1 puis sur F6.
  • Sélectionner Graph Type, puis appuyer sur F6 et F2. L’écran doit afficher Medbox.
  • Séries sans effectifs : sur Frequency, appuyer sur F1.
    Séries avec effectifs : sur Frequency, appuyer sur F2, 2 puis EXE.
  • Appuyer sur EXIT, puis sur F1.
